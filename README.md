# personal-finance-tracker
This is my personal finance tracker :) <br>
I track my finance with a spreadsheet, and I am trying to turn it into a Python desktop app with a wxPython based GUI. Currently this mainly displays data from CSV files, but I plan to add editing/saving functionalities and other features to the app, so I can eventually replace my spreadsheet tracker. <br>
